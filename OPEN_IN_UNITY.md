# Abrir no Unity

Este projeto foi criado para Unity `6000.5.3f1`.

No outro computador:

1. Extraia o zip.
2. Abra o Unity Hub.
3. Clique em `Add` / `Adicionar projeto do disco`.
4. Selecione a pasta que contem `Assets`, `Packages` e `ProjectSettings`.
5. Abra com Unity `6000.5.3f1` ou uma versao Unity 6 compativel.

O Unity vai recriar automaticamente as pastas `Library`, `Temp`, `Logs` e `UserSettings`.
