# Sons dos animais

Coloque os novos arquivos na pasta do animal correspondente em
`Assets/AnimalBattleRoyale/Resources/Audio/Animals`.

Use uma destas palavras no nome do arquivo:

- `basic`, `attack` ou `ataque`: ataque normal.
- `power`, `skill`, `ability`, `special` ou `habilidade`: habilidade.
- `hit`, `hurt`, `damage`, `impact` ou `dano`: golpe acertando o alvo.

Exemplos: `Tiger_basic_01.wav`, `Dog_power_01.wav`, `Penguin_hit_02.wav`.
Pode haver várias versões do mesmo evento; o jogo alterna entre elas e aplica uma pequena variação de pitch.
